NASA API Reference

We use the Mars Rover Photos API from api.nasa.gov. Core pattern:

GET https://api.nasa.gov/mars-photos/api/v1/rovers/{rover}/photos?sol={N}&camera={CAM}&api_key={KEY}

Examples (rover names are lowercase): curiosity, opportunity, spirit, and (where supported) perseverance. You can also use earth_date=YYYY-MM-DD instead of sol.
